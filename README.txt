The project contains 3 different types of turrets that use polymorphism to share some features.
TAB button changes between turrets.
2 types of enemies are spawned in the world.
Bullets and enemies are spawned at the start of the game. There's no actor spawn after begin play.
Most game parameters can be configured in the proper blueprints.
All the features were implemented in C++.
